This shows how sortable lists can be connected to each other.

Note the sortable CSS class (gives a min-height) so the lists never collapse when empty.

Super complicated example

1. This list cannot accept items

    • }
2. This is a normal list
    @for (item of list2; track item) {

    • }
3. This list clones its children
    @for (item of list3; track item) {
  • Element {{ item }}
    • }
